# FBLAViralFighter
A 2-D Sidescroller where you have to fend off viruses and collect motherboard components

You will want to run the Viral Showdown Remastered.exe file for the latest version

# Controls
A: Move

D: Move Right

W: Jump

T: Shoot

P: Pause

Esc: Exit to Desktop
